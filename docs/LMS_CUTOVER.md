# Переход на LMS (MVP)

## Маршруты

| Путь | Описание |
|------|----------|
| `/` | Вход / после входа перенаправление на `/lms` |
| `/lms` | Список назначенных курсов (learner) |
| `/lms/enrollment/:id` | Прохождение курса: теория, тест, практика |
| `/lms/admin/courses` | Админ/тьютор: курсы, модули, элементы, вопросы |
| `/lms/admin/org` | Админ/тьютор: организация, группы, назначения |
| `/lms/admin/practice` | Админ/тьютор: проверка практических работ |
| `/lms/analytics` | Сводка для администратора, тьютора, руководителя, наставника |
| `/course-legacy` | Прежний одно-курсовый интерфейс (21 день), `Dashboard` |

## База данных

Примените миграции сервера, включая `0014_lms_mvp` (таблицы `lms_*`, роли `learner` и т.д.):

```bash
cd server
npm run db:migrate
```

## API

Новые эндпоинты с префиксом `/api/lms/*` (см. `server/src/routes/lms.ts`).

## Роли

Целевые роли MVP: `admin`, `tutor`, `learner`, `department_head`, `mentor`.  
Глобальный флаг в `users` дополняется строками в `user_roles` для нескольких ролей.

## AI-тьютор

Исторический AI-квиз в старом курсе не является официальной аттестацией. Формальные тесты — только через блок **test** в LMS (`/lms/enrollment/...`).
